# Python Trivia Quiz Application

A modern, dark-themed Graphical User Interface (GUI) quiz application built with Python and Tkinter. This interactive application features dynamic score tracking, real-time feedback, and a premium visual aesthetic.

## 🚀 Features

* **Interactive Welcome Screen:** A clean splash page featuring quick details to onboard users before starting.
* **Modern Dark-Themed GUI:** Built with a premium custom dark palette (`#121218`), styled fonts, and interactive button hover effects.
* **Dynamic Score Tracking:** Real-time feedback displays the user's progress and current score at the top of every question card.
* **Smart Input Layout:** An integrated submission area accepting keyboard shortcuts (`Enter` key) for smooth, fluid gameplay.

## 🧠 Key Learnings

Building this project helped improve my understanding of:
* Object-Oriented Programming (OOP) in Python by structuring applications with classes (`tk.Tk`)
* Graphical User Interface (GUI) architecture, layout management (`pack`), and widget state manipulation
* Event binding handles for dynamic user actions (`<Enter>`, `<Leave>`, and `<Return>`)
* State tracking variables and dynamic UI rendering based on memory array items

